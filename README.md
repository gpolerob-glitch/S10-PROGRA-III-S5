# Editor de Texto con Stack (Undo/Redo)

## Descripción

Aplicación desarrollada en Python con Tkinter que simula un editor de texto. Permite escribir, borrar caracteres, deshacer y rehacer acciones utilizando la estructura de datos Stack (Pila).

 Funcionalidades

* Escribir texto.
* Borrar caracteres.
* Deshacer (Undo).
* Rehacer (Redo).
* Mostrar historial de acciones.
* Validación de errores.

Tecnologías Utilizadas

* Python 3
* Tkinter
* Programación Orientada a Objetos

## Ejecución

Ejecutar el archivo:

```bash
python main.py
```
